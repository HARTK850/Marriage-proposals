ברוך הבא ל-Yemot AI Ultimate

מערכת זו פותחה כדי לאפשר ניהול מלא ופיתוח אוטומטי של מערכות ימות המשיח. כאן תוכל:

  • לנהל את כל הקבצים והשלוחות במערכת שלך.
  • לגשת למאגר המידע העצום של ימות המשיח (הודעות מערכת והגדרות שונות).
  • להשתמש במעבדת ה-AI האוטונומית: לבקש מהבינה המלאכותית לכתוב קוד API, לפרוס אותו ל-Vercel דרך GitHub, ולהגדיר את השלוחה בימות המשיח - הכל בלחיצת כפתור אחת!

כדי להתחיל להשתמש בפריסה אוטונומית, ודא שהגדרת את חיבורי ה-GitHub וה-Gemini שלך בלשונית "הגדרות וחיבורים".

מה נבנה היום?

תאר את הלוגיקה (לדוגמה: תבנה API שבודק אם המספר מזוהה ומשמיע הודעה), והמערכת תכתוב את הקוד, תפרוס אותו ותגדיר את השלוחה.
